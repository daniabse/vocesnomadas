# Voces Nómadas

Web HTML estàtica amb relats de viatge, articles i cròniques del projecte *Voces Nómadas*.

El repo recull el lloc publicat a GitHub Pages, amb pàgines sobre rutes, experiències i contingut relacionat amb Àfrica i altres viatges.

## Què hi trobaràs

- Pàgina principal del projecte
- Articles i relats organitzats per categories i tags
- Pàgines informatives com `quienes-somos` i `contacto`

## Objectiu

Compartir històries de viatge i donar forma web al projecte de Henard i Dani.
